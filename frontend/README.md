# Frontend for HR Full Stack App

## Run locally

1. Start the backend from `backend_project`:
   - `npm run dev`

2. Start the frontend from `frontend`:
   - `cd frontend`
   - `npm install`
   - `npm run dev`

3. Open the frontend at `http://localhost:3000`.

## Features

- Login using `POST /api/auth/login`
- Dashboard with employee stats
- Employee list and add employee form
- Leave requests list and apply leave form
- Approve or reject pending leave requests
